# Mr.Brian Design Website

## Description

This repository contains the code for the Mr.Brian Design website. It showcases a variety of creative projects, including logo design, pixel art, educational design, AR/XR experiences, 3D modeling, and web3 projects. The site is designed with a bento grid layout for a visually engaging and responsive experience.

## Features

*   **Bento Grid Layout:** A visually appealing and responsive grid layout to showcase projects.
*   **Interactive Hover Effects:** Engaging hover effects on grid items.
*   **Animated Sections:** Sections animate into view on scroll.
*   **Navigation:** A fixed navigation bar for easy access to different sections.
*   **Footer:** A consistent footer with social media links.
*   **Responsive Design:** The website is designed to be responsive and work well on different screen sizes.
*   **Section Click Handling:** Clickable sections that navigate to dedicated pages.

## Technologies Used

*   **HTML:** The structure and content of the website.
*   **CSS:** Styling and layout, including custom properties (CSS variables), flexbox, and grid.
*   **JavaScript:** Interactive elements, animations, dynamic content loading.
*   **Git:** Version control.

## Setup Instructions

1.  **Clone the repository:**

    ```bash
    git clone <repository_url>
    ```

    Replace `<repository_url>` with the actual URL of your repository.
2.  **Navigate to the project directory:**

    ```bash
    cd <project_directory>
    ```

    Replace `<project_directory>` with the name of the directory you cloned the repository into.
3.  **Open `index.html` in your browser:**

    Simply open the `index.html` file in your web browser to view the website. No server is required for basic viewing.

## Customization

*   **Content:** Modify the HTML files (e.g., `index.html`, `logo.html`, etc.) to update the content of the website with your own projects and information.
*   **Styling:** Customize the appearance of the website by modifying the CSS files in the `src/css/` directory.
*   **JavaScript:** Extend the functionality of the website by adding or modifying JavaScript code in the `src/js/` directory.
